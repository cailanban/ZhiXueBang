# 需求文档

## 1. 应用概述

### 1.1 应用名称
智学帮 AI 学习系统

### 1.2 应用描述
面向高校学生的 AI 教育平台，以《Java 语言程序设计》为教学主线，覆盖 Java OOP、数据结构、C 语言、英语四级等学习场景。基于大模型和 12 智能体协同架构，提供智能对话、诊断型 AI 辅导、学习画像、知识库检索、多模态资源生成、学习路径规划、课程管理、练习评估、学习笔记、错题本、PPT 导出等核心能力。

### 1.3 项目定位
- 高校 AI 教育平台
- 主要学习场景：以《Java 语言程序设计》为主线，覆盖 Java、C 语言、数据结构、英语四级等
- 基于大模型和 12 智能体协同架构

### 1.4 视觉风格定位
参考 https://zhmt88.com/zh-CN 的视觉风格，打造科技感、未来感强烈的深色主题界面：
- 整体风格：深色主色调（深灰/黑色背景），科技感、未来感强烈
- 色彩体系：深色背景+渐变强调色（蓝紫渐变 #4F46E5 → #7C3AED），白色文字，卡片使用半透明深色背景
- 深色背景：#0a0b0d（更深）到 #111215
- 卡片设计：圆角卡片，带边框高亮效果，hover时有微妙光晕/发光效果
- 渐变效果：标题文字使用渐变色，按钮使用渐变背景
- 整体质感：高端、现代、科技感、视觉冲击力强

## 2. 用户与使用场景

### 2.1 目标用户
- 学习者：高校学生，需要系统化学习编程等技能
- 教师：负责课程管理、学习指导、评估反馈
- 管理员：负责系统配置、用户管理、数据统计

### 2.2 核心使用场景
- 学习者通过智能对话获取学习建议和知识解答
- 学习者使用诊断型AI辅导进行深度概念理解和知识漏洞诊断
- 学习者查看个人学习画像，了解薄弱点并获取改进建议
- 学习者使用知识库检索相关知识点，上传学习资料
- 学习者生成思维导图、代码案例、练习题等学习资源
- 学习者按照学习路径完成阶段性学习任务
- 学习者浏览课程列表，学习章节内容，完成练习题
- 学习者生成PPT课件辅助学习
- 学习者创建学习笔记，记录学习心得
- 学习者查看错题本，复习答错的练习题
- 学习者导出学习记录报告，分析学习数据
- 学习者切换暗亮模式，选择舒适的视觉体验
- 教师管理课程内容，查看学生学习进度
- 管理员管理用户、配置系统、查看数据统计

├── /tutor - 诊断型AI辅导页
├── /notes - 学习笔记页
├── /mistakes - 错题本页

### 3.2 页面功能说明

#### 3.2.1 Landing Page（/）
- Hero区域：大号标题+副标题描述，展示系统核心价值，带统计数字卡片展示（如「500+知识点」「10K+学习者」），居中布局
- 动态滚动：顶部品牌模型Logo横向滚动展示（如 DeepSeek V4、个人知识库等）
- 产品特性展示：编号（01/02/03）的功能特性列表，每个特性有大图和说明（智能对话、诊断型辅导、学习画像、知识库、资源生成、学习路径等）
- 智能体展示：展示十二智能体能力（对话、诊断、画像、知识库、路径规划、课程架构、讲义编写、思维导图、题库出题、代码案例、资源推荐、质量审核）
- 数据统计：展示系统核心数据（课程数、学习者数、知识点数等）
- 导航：深色顶部导航栏，Logo+菜单+CTA按钮（登录/注册）
- 视觉风格：深色主题，蓝紫渐变强调色，圆角卡片，hover光晕效果

#### 3.2.2 登录/注册页（/login）
- 注册功能：用户输入邮箱、密码完成注册，选择角色（学习者/教师/管理员）
- 登录功能：用户输入邮箱、密码完成登录，使用login skill实现JWT认证
- 隐私政策弹窗：首次访问时弹出隐私政策，用户同意后方可继续使用（C01合规要求）
- 密码找回：通过邮箱验证找回密码

#### 3.2.3 学习中心首页（/dashboard）
- 数据概览：展示学习时长、完成课程数、练习题完成率、学习画像评分等关键数据
- 快捷入口：提供智能对话、诊断型AI辅导、学习画像、知识库、资源生成、学习笔记、错题本等功能入口
- 最近学习：展示最近学习的课程和章节
- 学习提醒：展示待完成任务和学习建议
- 暗亮模式切换：顶部导航栏右侧提供太阳/月亮图标切换按钮

#### 3.2.4 智能对话页（/chat）
- 多模态输入：支持文本和图片输入，调用大模型（DeepSeek V4 / 讯飞星火 Spark Pro）
- 实时流式输出：使用SSE技术实现AI回复的实时流式展示
- 对话历史：展示历史对话记录，支持上下文管理
- 快捷指令面板：提供生成路径、构建画像、生成资源等快捷指令
- 知识库检索：对话中自动调用ima-knowledge skill检索相关知识点
- 智能体执行轨迹：可视化展示每个Agent的执行过程

#### 3.2.5 诊断型AI辅导页（/tutor）
- 学习模式选择面板：提供零基础模式、普通模式、进阶模式、Auto模式四种学习模式选择
- 学科诊断结果卡片：展示当前学习主题的诊断结果，包括学科领域→知识系统→子主题→核心概念→前置知识的层级结构
- 对话区域：支持文本输入，AI基于universal-diagnostic-tutor规则进行诊断型教学，支持Markdown和LaTeX数学公式渲染（KaTeX）
- Mermaid图解渲染：自动检测AI回复中的Mermaid代码块并渲染为流程图、架构图、关系图
- 教学深度自适应：Level 1-5自适应调整，从一句话答案到完整知识体系讲解
- 教-检-进节奏：每次教一个小知识点，检查理解后再继续
- 知识漏洞诊断：识别词汇、概念、符号、步骤、推理、迁移等类型的知识缺口
- 知识掌握状态追踪：在当前会话内追踪学习状态，展示已掌握和待掌握的知识点
- 快速练习题面板：根据当前学习主题生成针对性练习题
- 学习状态卡生成：生成可复制的Learning State Card，方便跨对话继续学习
- 代码块语法高亮：支持代码块语法高亮显示
- 适用学科：大学STEM课程（数学、编程、算法、AI/ML、物理、电子、信号）

#### 3.2.6 学习笔记页（/notes）
- 笔记列表视图：展示所有笔记，按时间/分类排序
- 创建笔记：点击创建按钮，打开富文本编辑器，输入笔记内容
- 编辑笔记：点击笔记卡片进入笔记详情页，支持编辑
- 删除笔记：在笔记列表或详情页删除笔记
- 笔记分类标签管理：为笔记添加自定义标签，支持按标签筛选
- 笔记搜索：全文检索笔记内容
- 笔记与课程章节关联：从课程详情页打开笔记，自动关联当前章节
- 笔记详情页：支持Markdown渲染
- 一键导出笔记：导出为TXT或Markdown格式

#### 3.2.7 错题本页（/mistakes）
- 错题列表：展示所有答错的练习题
- 自动收录：用户答错练习题后，系统自动加入错题本
- 手动添加：用户可手动添加题目到错题本
- 筛选功能：按课程/章节/难度筛选错题
- 错题复习模式：逐题翻看+解析
- 标记状态：标记已掌握/未掌握状态
- 错题本统计：展示各类错误率分析
- 批量操作：删除已掌握、按章节清空

#### 3.2.8 学习画像页（/profile）
- 六维画像展示：以雷达图形式展示知识基础、认知风格、学习偏好、易错点、学习目标、学习节奏六个维度
- 画像更新：基于学习行为数据自动更新画像
- 历史追踪：查看画像历史变化趋势
- 薄弱点分析：展示当前薄弱点及改进建议

#### 3.2.9 知识库管理页（/knowledge）
- 资料上传：上传学习资料到个人知识库，使用ima-knowledge skill
- 知识检索：输入关键词检索相关知识点
- 知识点详解：展示检索结果，包含知识点内容和来源标注
- 网页资源导入：使用web-reader导入网页资源到知识库
- 知识卡片展示：以卡片形式展示知识点摘要

#### 3.2.10 资源生成页（/resources）
- 资源类型选择：选择生成思维导图、代码案例、练习题、阅读清单、课程讲解

- 资源预览：预览生成的资源内容
- 资源下载：下载生成的资源文件
- 智能体执行轨迹：展示资源生成过程中各Agent的执行步骤
- PPT课件生成：输入学习主题，使用ppt-generator skill自动生成PPT大纲和完整课件，支持预览与下载
- 代码案例生成：输入知识点描述，生成可运行的 Java 代码示例

#### 3.2.11 学习路径页（/path）
- 四阶段路径展示：展示基础→进阶→实战→评估四个阶段
- 阶段详情：每个阶段包含学习目标、资源清单、验收标准、预计时间
- 推荐资源列表：展开阶段卡片时显示推荐资源（讲义、思维导图、练习题等），点击跳转ResourcesPage并预填主题

#### 3.2.12 课程列表页（/courses）
- 课程展示：展示Java、C语言、数据结构、英语四级等课程列表
- 搜索与筛选：按课程名称、分类、难度筛选课程

- 练习题：展示章节配套练习题（选择题、填空题、编程题）

- 笔记入口：从章节学习页打开笔记，自动关联当前章节

#### 3.2.14 学习评估报告页（/evaluation）

## 4. 业务规则与逻辑

### 4.1 用户体系与权限
- 用户注册后默认为学习者角色，管理员可修改用户角色
- 学习者可访问学习相关功能，不可访问管理后台
- 教师可访问课程管理和学生学习数据
- 管理员可访问所有功能
- 使用JWT认证，token有效期7天

### 4.2 隐私合规
- 用户首次访问时必须同意隐私政策方可继续使用
- 隐私政策和用户协议在个人设置页提供双清单入口
- 用户可申请账号注销，注销后删除所有个人数据

### 4.3 AI能力调用
- 所有AI能力调用均通过后端Edge Functions执行，API密钥不暴露前端
- 支持DEMO模式：当API不可用时展示示例数据，不影响用户体验
- 智能对话使用 DeepSeek V4 大模型，支持流式输出
- 诊断型 AI 辅导通过 Edge Function（deepseek-tutor / spark-tutor）调用大模型（multimodal-llm skill），系统提示词注入universal-diagnostic-tutor的诊断型教学规则，并增加指令：当解释涉及流程、架构、数据结构、关系时，在回复末尾附加Mermaid代码块
- 知识库检索使用个人知识库，支持上传资料和检索
- 资源生成使用大模型生成文本资源
- PPT生成使用generate-ppt Edge Function

### 4.4 诊断型AI辅导规则
- 学科诊断流程：先判断学科领域→知识系统→子主题→核心概念，再开始教学
- 知识漏洞诊断类型：词汇、概念、符号、步骤、推理、迁移
- 学习模式：零基础模式、普通模式、进阶模式、Auto模式
- 教学深度：Level 1-5自适应调整
- 教学节奏：教-检-进，每次教一个小知识点，检查理解后再继续
- 学习状态追踪：在当前对话内追踪学习状态，不依赖持久记忆
- 学习状态卡：生成可复制的Learning State Card，方便跨对话继续
- Mermaid图解：当解释涉及流程、架构、数据结构、关系时，在回复末尾附加Mermaid代码块，前端自动检测并渲染

### 4.5 /chat与/tutor的区别
- /chat：通用AI对话，支持多模态输入，RAG知识库检索，资源生成，快捷指令
- /tutor：诊断型教学辅导，聚焦深度理解和掌握，有明确的教学节奏和知识漏洞诊断，支持Mermaid图解，适用于大学STEM课程（数学、编程、算法、AI/ML、物理、电子、信号）

### 4.6 学习画像更新
- 学习画像基于用户学习行为数据自动更新（学习时长、练习完成率、答题正确率等）
- 画像更新频率：每完成一个章节学习或每完成10道练习题后更新
- 画像历史记录保留最近30天数据

### 4.7 学习路径调整
- 学习路径初始生成基于用户选择的课程和学习目标
- 完成阶段评估后，根据评估结果动态调整后续阶段内容
- 调整规则：薄弱点对应的知识点增加练习资源，已掌握的知识点减少重复内容
- deepseek-path生成时包含recommendedResources字段，每个step包含推荐资源列表（类型、主题）
- 阶段完成度变化超过20%时，toast提示「学习进度已更新，是否重新规划后续路径？」

### 4.8 练习题反馈
- 用户答题后，AI提供即时反馈
- 反馈方式：逐步提示，先提示思路方向，再提示关键知识点，最后给出完整解析
- 错题自动加入错题本
- 答题数据同步写入quiz_attempts表（user_id, course_id, topic_id, score, correct, total, attempted_at）

### 4.9 学习笔记规则
- 笔记支持富文本编辑，支持Markdown渲染
- 笔记可关联课程章节，从课程详情页打开笔记时自动关联当前章节
- 笔记支持自定义标签分类
- 笔记支持全文检索
- 笔记可导出为TXT或Markdown格式

### 4.10 错题本规则
- 用户答错练习题后，系统自动加入错题本
- 用户可手动添加题目到错题本
- 错题本支持按课程/章节/难度筛选
- 错题本支持标记已掌握/未掌握状态
- 错题本支持批量操作（删除已掌握、按章节清空）

### 4.11 学习记录数据导出规则
- 支持导出CSV和PDF格式
- 支持时间范围筛选（最近7天/30天/全部）
- 导出内容包含：学习时长统计、课程完成情况、练习题答题情况、学习画像变化

### 4.12 暗亮模式切换规则
- 全局暗色/亮色主题切换
- 切换按钮放在顶部导航栏右侧（太阳/月亮图标）
- 用户偏好持久化存储（localStorage）
- 亮色模式：白色背景+深色内容
- 暗色模式：深灰背景（#1a1b1e）+亮色文字
- 两种模式下都保持完整的视觉层次感
- 默认主题为暗色模式

### 4.13 数据存储与安全
- 使用Supabase数据库存储用户数据、课程数据、学习记录、笔记数据、错题数据、答题数据（quiz_attempts表）
- 使用Supabase Auth管理用户认证
- 使用RLS（Row Level Security）权限控制，确保用户只能访问自己的数据
- 防护SQL注入和XSS攻击

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户未同意隐私政策 | 阻止访问系统，持续展示隐私政策弹窗 |
| 登录token过期 | 自动跳转登录页，提示用户重新登录 |
| AI能力调用失败 | 展示友好错误提示，启用DEMO模式展示示例数据 |
| 知识库上传失败 | 提示用户重新上传，记录失败日志 |
| 用户无权限访问页面 | 跳转403页面，提示权限不足 |
| 网络异常 | 展示网络异常提示，提供重试按钮 |
| 课程内容为空 | 提示暂无内容，引导用户浏览其他课程 |
| 学习画像数据不足 | 提示需完成更多学习任务后才能生成完整画像 |
| 管理员删除正在学习的课程 | 学习者端展示课程已下架提示 |
| 诊断型辅导学科不支持 | 提示当前学科暂不支持诊断型辅导，引导用户使用智能对话功能 |
| LaTeX公式渲染失败 | 展示原始LaTeX代码，提示渲染异常 |
| Mermaid代码块渲染失败 | 展示原始Mermaid代码，提示渲染异常 |
| 笔记保存失败 | 提示用户保存失败，允许重新保存 |
| 错题本为空 | 提示暂无错题，引导用户完成练习题 |
| 数据导出失败 | 提示用户导出失败，允许重新导出 |
| quiz_attempts表写入失败 | 记录失败日志，不影响答题流程 |

## 6. 验收标准

1. 用户访问Landing Page（/），查看Hero区域、功能特性展示、智能体展示、数据统计，点击登录按钮
2. 用户在登录页输入邮箱和密码完成注册，同意隐私政策后进入系统
3. 用户在学习中心首页查看学习数据概览，点击诊断型AI辅导入口
4. 用户在诊断型AI辅导页选择普通模式，输入学习问题（如「什么是Java多态」），AI进行学科诊断并展示诊断结果卡片
5. AI基于教-检-进节奏进行教学，用户回答检查问题后继续学习下一个知识点
6. 用户输入「解释TCP三次握手的流程」，AI回复末尾附加Mermaid流程图，前端自动渲染
7. 用户在知识掌握状态追踪面板查看当前会话内已掌握和待掌握的知识点
8. 用户生成Learning State Card并复制，退出后重新进入诊断型AI辅导页，粘贴Learning State Card继续学习
9. 用户在智能对话页输入学习问题（文本或图片），AI实时流式输出回复
10. 用户点击快捷指令「构建画像」，系统生成六维学习画像并展示雷达图
11. 用户在知识库管理页上传学习资料，检索相关知识点并查看详解
12. 用户在课程列表页选择Java课程，进入课程详情页学习第一章节
14. 用户点击知识点旁的「查看代码」按钮，系统展示 Java 代码示例
15. 用户完成章节练习题，AI提供即时反馈，答错的题目自动加入错题本，答题数据同步写入quiz_attempts表
16. 用户从课程详情页打开笔记，创建学习笔记并关联当前章节，添加自定义标签
17. 用户在错题本页查看答错的练习题，进入错题复习模式，标记已掌握状态
18. 用户在学习路径页查看四阶段学习路径，展开阶段卡片查看推荐资源列表，点击跳转ResourcesPage并预填主题
19. 用户完成基础阶段后进入进阶阶段，系统检测到阶段完成度变化超过20%，toast提示「学习进度已更新，是否重新规划后续路径？」
20. 用户在学习评估报告页查看评估结果和知识薄弱点标签云（错误率最高的5个知识点），导出学习记录报告（CSV或PDF格式）
21. 用户在个人设置页切换暗亮模式，系统保存用户偏好
22. 用户在个人设置页申请账号注销，系统删除所有个人数据
